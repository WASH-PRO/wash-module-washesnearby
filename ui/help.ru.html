Автомойки рядом

Отправляет мойки CRM на публичную карту через /api/v1/integration/*.

Настройка

  1. Возьмите Owner API токен в кабинете владельца на сайте.
  2. Укажите адрес сайта / API, интервал отправки, лимит новостей и часовой пояс кассы.
  3. В блоке «Мойки» задайте координаты, город и тип для каждой мойки CRM.
  4. Сохраните и запустите модуль.

Касса (финансы)

В каждом telemetry (раз в ≥ 60 с) модуль читает /api/crm/finance-stats и добавляет блок finance: наличные / внешние (безнал) / скидки за сегодня, до и после инкассации — по мойке и по постам.

  • before_collection / after_collection — живые счётчики CRM (cash, cashless→external, discountOps→discount).
  • today — прирост накопительных after_collection за календарный день в выбранной таймзоне.
  • Периоды 7 дней и 30 дней в кабинете владельца считает сайт (сумма дневных today) — модуль их не шлёт.

Цены

Для каждого кода режима — максимальная цена среди постов мойки. Названия — из /api/crm/work-modes.

Занятость

Telemetry с постами уходит каждый цикл (≥ 60 с). Цены — через PATCH только при изменении контента.

Новости и акции

Из «Публикаций» CRM: news/general → новости, promotion → акции.
