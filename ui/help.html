Washes Nearby

Pushes CRM washes to the public map via /api/v1/integration/*.

Setup

  1. Get an Owner API token in the Washes Nearby owner cabinet.
  2. Set default coordinates (CRM stores address text only).
  3. Optional: paste existing CRM→site ids into wash_mapping.
  4. Start the module.

Prices

For each work mode code, the module reads posts.settings.modePrices on every post of the wash and sends the maximum price. Mode names come from /api/crm/work-modes.

Occupancy

Telemetry sends post status every cycle (≥ 60 s). Prices are updated via PATCH only when content changes, to avoid re-moderation every minute.

News & promotions

From CRM Publications: news/general → news, promotion → promotions. Only scheduled/published items with publishedAt in the past.
